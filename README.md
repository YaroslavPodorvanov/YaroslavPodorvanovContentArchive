# Yaroslav Podorvanov Content Archive

This repository is a public raw archive of my writing from various external platforms.

It is NOT a blog, NOT a curated knowledge base, and NOT a publishing system.

## Purpose

The purpose of this repository is to:
- Preserve raw copies of my written content from platforms such as Reddit and others
- Provide a public backup of text content in Markdown format
- Maintain transparency and availability of previously published material

## Structure

All content is stored in the `raw/` directory, grouped by source platform:
raw/
&nbsp;&nbsp;&nbsp;&nbsp;reddit/

Each file typically represents a single post or comment exported in Markdown format.

## Important Notes

- Content is stored as-is, without editorial changes
- This repository is not optimized for reading experience or SEO
- No content is actively maintained or updated
- This is an archive, not a blog

## Disclaimer

Content reflects the context and understanding at the time it was originally written.
